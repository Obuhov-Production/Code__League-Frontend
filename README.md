# Code League – Fronten

У цьому репозиторії представлена частина **Frontend** проєкту Code League.

Наразі реалізовано:

* Привітальна сторінка (Landing Page) з основною інформацією про платформу
* Інтеграція з Backend для виклику функцій
* Авторизація
* Контактні форми
* Базова взаємодія з серверною частиною

## 🚀 Технології Frontend

* **React 19.2** — бібліотека для побудови користувацьких інтерфейсів
* **Vite 7.3** — швидкий інструмент для розробки та збірки
* **ESLint** — контроль якості та стилю коду

## 📦 Встановлення

```bash
npm install
```

## Запуск проєкту

### Режим розробки (з автоматичним перезавантаженням)

```bash
npm run dev
```

### Збірка для продакшену

```bash
npm run build
```

### Перегляд зібраної версії

```bash
npm run preview
```

### Перевірка коду

```bash
npm run lint
```


## Структура проєкту

```
src/
├── App.jsx              # Головний компонент
├── main.jsx             # Точка входу
└── assets/
    ├── images/          # Зображення
    │   ├── company/
    │   ├── components/
    │   ├── favicon/
    │   ├── main/
    │   └── other/
    ├── scripts/         # JavaScript/TypeScript файли
    │   ├── API/         # Запити до Backend
    │   └── Frontend/    # Логіка Frontend
    └── styles/          # Стилі
        ├── main.css
        ├── adaptive/    # Адаптивні стилі
        ├── animation/   # Анімації
        ├── components/  # Стилі компонентів
        ├── layouts/     # Стилі макетів
        └── main_design/ # Основний дизайн
```
