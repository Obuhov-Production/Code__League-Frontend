# Code League – frontend

У цьому репозиторії представлена частина **Frontend** проєкту Code League.

## 🚀 Технології Frontend

* **React** — бібліотека для побудови користувацьких інтерфейсів
* **Vite** — швидкий інструмент для розробки та збірки
* **ESLint** — контроль якості та стилю коду

## 📦 Встановлення
```bash
npm install
```

## Запуск проєкту
### Режим розробки
```bash
npm run dev
```

### Білдік
```bash
npm run build
```

### Перегляд зібраної версії
```bash
npm run preview
```

### Перевірка коду
```bash
npm run lint
```
